# 🏠 选房对比 / House Compare

双语（中/英）房屋对比工具，支持大文件上传、自动保存、可分享只读页面。

## ✨ 功能特性

- 🌐 **中英双语** — 一键切换界面语言
- 💾 **实时自动保存** — 使用 IndexedDB，刷新/关闭浏览器数据不丢失
- 📹 **大文件支持** — 可上传几十MB的视频和图片，存储在浏览器 IndexedDB 中
- 🔗 **导出可分享页面** — 生成独立 HTML 文件，所有媒体内嵌，发给别人即可查看（只读）
- 📦 **JSON 导入/导出** — 备份和恢复数据
- 📱 **响应式设计** — 手机/平板/桌面都好用
- 🚀 **纯静态** — 可直接部署到 GitHub Pages

## 🚀 部署到 GitHub Pages

### 方法一：Fork 本仓库

1. Fork 这个仓库
2. 进入 Settings → Pages
3. Source 选 `main` 分支，目录选 `/ (root)`
4. 保存后几分钟内即可访问 `https://你的用户名.github.io/仓库名/`

### 方法二：手动上传

1. 创建新仓库
2. 上传 `index.html`
3. 启用 GitHub Pages（同上步骤）

### 方法三：命令行部署

```bash
# 克隆或初始化仓库
cd house-compare
git init
git add index.html README.md
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main

# 然后去 GitHub 设置 Pages
```

## 📖 使用说明

1. **添加选项** — 点击「＋ 添加选项」按钮
2. **展开详情** — 点击「区域详情」展开卡片
3. **上传图片** — 点击图片区域的 `+` 号
4. **上传视频** — 展开后点击「📁 上传视频」或「🔗 添加链接」
5. **添加区域** — 默认有卧室/客厅/厨房等，可自定义添加
6. **自动保存** — 所有操作自动保存到浏览器，无需手动保存
7. **分享** — 点击「🔗 导出可分享页面」生成 HTML 文件，发给别人即可查看

## 🔒 隐私说明

- 所有数据保存在**你的浏览器本地**（IndexedDB）
- 导出的 HTML 文件包含所有媒体数据，文件可能较大
- 不会上传任何数据到服务器
- 清除浏览器数据会丢失所有内容，建议定期导出 JSON 备份

## 💡 提示

- 视频文件会以 Base64 编码存储，大文件可能影响页面性能
- 建议单个视频不超过 100MB
- 图片建议压缩后再上传以获得更好体验
- 导出的可分享 HTML 文件大小 = 所有媒体文件大小之和
